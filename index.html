Visão Geral

Bem-vindo à documentação do plugin AI Content Optimizer . Este plugin foi desenvolvido para ajudar você a otimizar o conteúdo do seu site WordPress, incluindo posts, produtos, páginas e Custom Post Types (CPTs), utilizando o poder da inteligência artificial Gemini para fornecer sugestões de SEO, títulos, subtítulos, descrições e palavras-chave.

Nosso objetivo é simplificar o processo de otimização de conteúdo, permitindo que você melhore o ranking do seu site nos motores de busca e atraia mais visitantes de forma eficiente.

Overview

Welcome to the documentation for the AI Content Optimizer plugin. This plugin was developed to help you optimize your WordPress website content, including posts, products, pages, and Custom Post Types (CPTs), using the power of Gemini artificial intelligence to provide suggestions for SEO, titles, subtitles, descriptions, and keywords.

Our goal is to simplify the content optimization process, allowing you to improve your website's search engine rankings and attract more visitors efficiently.

1. Funcionalidades Principais

  • Análise de Conteúdo com IA (Gemini): Analisa o conteúdo existente do seu post, página ou CPT para identificar oportunidades de otimização.
  • Sugestões de Palavras-chave: Receba sugestões de palavras-chave relevantes com base no seu conteúdo.
  • Sugestões de Títulos, Subtítulos e Descrições: Obtenha ideias para melhorar seus títulos, subtítulos e meta descrições para atrair mais tráfego.
  • Dashboard Intuitivo: Uma interface centralizada no painel de administração para gerenciar e otimizar todo o seu conteúdo.
  • Aplicação Direta de Sugestões: Aplique as sugestões de títulos, subtítulos, descrições e palavras-chave diretamente do dashboard ou da modal de sugestões.
  • Histórico de Análises: Acompanhe o histórico de análises realizadas para cada item de conteúdo.
  • Suporte a Posts, Páginas, Produtos e CPTs: Otimize diversos tipos de conteúdo no seu site.

1. Key Features

  • AI Content Analysis (Gemini): Analyzes the existing content of your post, page, or CPT to identify optimization opportunities.
  • Keyword Suggestions: Receive relevant keyword suggestions based on your content.
  • Title, Subtitle, and Description Suggestions: Get ideas to improve your titles, subtitles, and meta descriptions to attract more traffic.
  • Intuitive Dashboard: A centralized interface in the admin panel to manage and optimize all your content.
  • Direct Application of Suggestions: Apply title, subtitle, description, and keyword suggestions directly from the dashboard or the suggestions modal.
  • Analysis History: Track the history of analyses performed for each content item.
  • Support for Posts, Pages, Products, and CPTs: Optimize various types of content on your website.

2. Instalação

Para instalar o plugin "AI Content Optimizer ", siga os passos abaixo:

2.1 Upload via WordPress

  1. Faça o download do arquivo .zip do plugin.
  2. No painel de administração do WordPress, navegue até Plugins > Adicionar Novo.
  3. Clique no botão Enviar Plugin.
  4. Clique em Escolher arquivo e selecione o arquivo .zip que você baixou.
  5. Clique em Instalar Agora.
  6. Após a instalação, clique em Ativar Plugin.

2.2 Upload via FTP

  1. Faça o download do arquivo .zip do plugin e extraia-o para uma pasta no seu computador.
  2. Conecte-se ao seu servidor via FTP.
  3. Navegue até o diretório wp-content/plugins/.
  4. Faça o upload da pasta extraída do plugin para este diretório.
  5. No painel de administração do WordPress, navegue até Plugins > Plugins Instalados.
  6. Encontre "AI Content Optimizer " na lista e clique em Ativar.

2. Installation

To install the "AI Content Optimizer " plugin, follow the steps below:

2.1 Upload via WordPress

  1. Download the plugin's .zip file.
  2. In the WordPress admin panel, navigate to Plugins > Add New.
  3. Click the Upload Plugin button.
  4. Click Choose File and select the .zip file you downloaded.
  5. Click Install Now.
  6. After installation, click Activate Plugin.

2.2 Upload via FTP

  1. Faça o download do arquivo .zip do plugin e extraia-o para uma pasta no seu computador.
  2. Conecte-se ao seu servidor via FTP.
  3. Navegue até o diretório wp-content/plugins/.
  4. Upload the extracted plugin folder to this directory.
  5. In the WordPress admin panel, navigate to Plugins > Installed Plugins.
  6. Find "AI Content Optimizer " in the list and click Activate.

3. Configuração

Após a instalação e ativação, você precisará configurar a chave da API Gemini:

  1. No painel de administração do WordPress, navegue até AI Optimizer > Settings.
  2. Localize o campo para a Chave da API Gemini.
  3. Insira sua chave da API Gemini válida neste campo.
  4. Clique em Salvar Alterações.

Como obter uma Chave da API Gemini:

Você pode obter uma chave da API Gemini através da Google AI for Developers. Siga as instruções na documentação oficial do Google para gerar sua chave.

3. Configuration

After installation and activation, you will need to configure the Gemini API key:

  1. In the WordPress admin panel, navigate to AI Optimizer > Settings.
  2. Locate the field for the Gemini API Key.
  3. Enter your valid Gemini API key in this field.
  4. Click Save Changes.

How to obtain a Gemini API Key:

You can obtain a Gemini API key through Google AI for Developers. Siga as instruções na documentação oficial do Google para gerar sua chave.

4. Uso

Acesse o dashboard do plugin para começar a otimizar seu conteúdo:

  1. No painel de administração do WordPress, navegue até AI Optimizer > AI Optimizer.
  2. Você verá abas para diferentes tipos de conteúdo (Publicações, Produtos, Páginas e CPTs). Clique na aba desejada.
  3. A tabela exibirá uma lista do seu conteúdo. Para cada item, você verá:
    • Conteúdo: O título do post, página ou CPT.
    • Tipo: O tipo de conteúdo (Post, Página, Produto, etc.).
    • Qtd: O número de análises já realizadas para este item.
    • Última Análise: A data e hora da última análise.
    • Status da Análise: O status atual da análise.
    • Ação: Botões para interagir com o conteúdo.

4.1 Botões de Ação

  • Botão Analisar (): Clique neste botão para iniciar uma nova análise de conteúdo para o item específico. O status na tabela será atualizado para "Analisando..." e, ao final, para "Analisado" ou "Erro na Análise".
  • Botão Ver Informações Atuais (): Clique neste botão para abrir uma modal exibindo as informações atuais do conteúdo (título, subtítulo, descrição, palavras-chave).
  • Botão Ver Histórico de Análises (): Clique neste botão para abrir uma modal exibindo o histórico de análises realizadas para este item de conteúdo.

4.2 Resultados da Análise

Após clicar no botão "Analisar", a seção "Resultados da Análise por IA" aparecerá abaixo das abas.

  • Coluna Esquerda: Exibe as "Palavras-chave Sugeridas" em uma tabela, incluindo a quantidade estimada de "Buscas/mês".
  • Coluna Direita: Exibe "Títulos, Subtítulos e Descrições" sugeridos em uma tabela, também com a quantidade estimada de "Buscas/mês".

4.3 Modal de Sugestões Detalhadas

Clique no botão 🚀 Gemini () na seção "Resultados da Análise por IA" para abrir a modal de sugestões detalhadas. Esta modal oferece:

  • Sugestões de Palavras-chave: Uma lista de palavras-chave sugeridas. Você pode clicar no botão "Utilizar Todas as Palavras-chave Sugeridas" para adicioná-las como tags ao seu post (se o tipo de conteúdo suportar tags). Palavras-chave já aplicadas serão destacadas.
  • Sugestões de Títulos: Uma tabela com títulos sugeridos, a quantidade de "Buscas/mês" e um botão "Usar" para aplicar o título diretamente ao seu post/página/CPT. Títulos já aplicados serão destacados.
  • Sugestões de Subtítulos: Uma tabela com subtítulos sugeridos, a quantidade de "Buscas/mês" e um botão "Usar" para aplicar o subtítulo. Subtítulos já aplicados serão destacados.
  • Sugestões de Descrição: Uma lista de descrições sugeridas com um botão "Usar esta Descrição" para aplicar a descrição ao conteúdo principal do seu post/página/CPT. Descrições que parecem já estar no conteúdo serão destacadas (basado em uma verificação simples dos primeiros 100 caracteres).

Ao clicar nos botões "Usar" ou "Utilizar Todas as Palavras-chave Sugeridas", o plugin fará uma requisição AJAX para atualizar o conteúdo do seu post/página/CPT no banco de dados.

4. Usage

Access the plugin dashboard to start optimizing your content:

  1. In the WordPress admin panel, navigate to AI Optimizer > AI Optimizer.
  2. You will see tabs for different content types (Posts, Products, Pages, and CPTs). Click on the desired tab.
  3. The table will display a list of your content. For each item, you will see:
    • Content: The title of the post, page, or CPT.
    • Type: The content type (Post, Page, Product, etc.).
    • Qty: The number of analyses already performed for this item.
    • Last Analysis: The date and time of the last analysis.
    • Analysis Status: The current status of the analysis.
    • Action: Buttons to interact with the content.

4.1 Action Buttons

  • Analyze Button (): Click this button to start a new content analysis for the specific item. The status in the table will update to "Analyzing..." and, upon completion, to "Analyzed" or "Analysis Error".
  • View Current Info Button (): Click this button to open a modal displaying the current content information (title, subtitle, description, keywords).
  • View History Button (): Click this button to open a modal displaying the history of analyses performed for this content item.

4.2 Analysis Results

After clicking the "Analyze" button, the "AI Analysis Results" section will appear below the tabs.

  • Left Column: Displays "Suggested Keywords" in a table, including the estimated "Searches/month".
  • Right Column: Displays suggested "Titles, Subtitles, and Descriptions" in a table, also with the estimated "Searches/month".

4.3 Detailed Suggestions Modal

Click the 🚀 Gemini () button in the "AI Analysis Results" section to open the detailed suggestions modal. This modal offers:

  • Keyword Suggestions: A list of suggested keywords. You can click the "Use All Suggested Keywords" button to add them as tags to your post (if the content type supports tags). Applied keywords will be highlighted.
  • Title Suggestions: A table with suggested titles, the number of "Searches/month", and a "Use" button to apply the title directly to your post/page/CPT. Títulos já aplicados serão destacados.
  • Subtitle Suggestions: A table with suggested subtitles, the number of "Searches/month", and a "Use" button to apply the subtitle. Subtítulos já aplicados serão destacados.
  • Description Suggestions: A list of descriptions suggested with a "Use This Description" button to apply the description to the main content of your post/page/CPT. Descriptions que parecem já estar no conteúdo serão destacadas (basado em uma verificação simples dos primeiros 100 caracteres).

Ao clicar nos botões "Usar" ou "Utilizar Todas as Palavras-chave Sugeridas", o plugin fará uma requisição AJAX para atualizar o conteúdo do seu post/página/CPT no banco de dados.

5. Perguntas Frequentes (FAQ)

  • O plugin funciona com qualquer tema? Sim, o plugin opera no painel de administração do WordPress e não interfere diretamente com o frontend do seu site ou com o tema utilizado.
  • Preciso ter conhecimentos de programação para usar o plugin? Não, o plugin é projetado para ser fácil de usar através da interface do painel de administração.
  • O plugin armazena minha chave da API Gemini de forma segura? Sim, a chave da API é armazenada nas configurações do WordPress e deve ser tratada com segurança.
  • O que acontece se a API Gemini retornar um erro? O plugin exibirá uma mensagem de erro no dashboard. Verifique sua chave da API e as configurações.
  • Posso analisar Custom Post Types? Sim, o plugin lista e permite analisar CPTs públicos registrados no seu site.

5. Frequently Asked Questions (FAQ)

  • Does the plugin work with any theme? Yes, the plugin operates in the WordPress admin panel and does not directly interfere with your website's frontend or the theme used.
  • Do I need programming knowledge to use the plugin? No, the plugin is designed to be easy to use through the admin panel interface.
  • Does the plugin store my Gemini API key securely? Yes, the API key is stored in WordPress settings and should be treated securely.
  • What happens if the Gemini API returns an error? The plugin will display an error message on the dashboard. Check your API key and settings.
  • Can I analyze Custom Post Types? Yes, the plugin lists and allows you to analyze public CPTs registered on your website.
